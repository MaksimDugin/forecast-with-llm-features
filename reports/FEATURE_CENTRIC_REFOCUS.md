## Новый основной фокус

Курсовая пишется не как работа про максимальное качество модели, а как работа про признаки:

```text
теоретическое обоснование LLM/FinBERT-generated features
+
проверка их влияния на прогноз финансового временного ряда
```

Модель в этой логике — инструмент проверки признаков, а не главный объект исследования.

---

## Что считать главным результатом

Главный результат — не accuracy LSTM сама по себе, а сравнение групп признаков:

1. `prices-only` — что даёт только рыночная динамика;
2. `FinBERT-only` — есть ли самостоятельный сигнал в текстовых embedding-признаках;
3. `prices + FinBERT` — дают ли текстовые признаки добавочную предсказательную силу поверх рыночных признаков;
4. `majority baseline` — минимальная точка отсчёта.

## Как нужно перестроить текст

### Введение

Акцентировать не «построение модели прогнозирования», а:

```text
оценку того, как признаки, извлечённые из финансовых текстов с помощью FinBERT, влияют на качество прогноза направления цены.
```

### Теоретическая часть

Сделать сильнее разделы:

1. что такое признаки в ML-задаче;
2. почему финансовые тексты могут быть источником признаков;
3. почему FinBERT embeddings теоретически могут нести рыночный сигнал;
4. чем отличаются sentiment-score и embeddings;
5. что значит добавочная предсказательная сила признаков.

### Экспериментальная часть

Модели описывать короче, как инструменты проверки feature groups.

Главные сравнения:

```text
FinBERT-only vs majority
prices + FinBERT vs prices-only
prices + FinBERT vs FinBERT-only
LSTM vs logistic regression
```

### Результаты

Результаты интерпретировать по признакам:

- FinBERT-only показывает слабый самостоятельный сигнал;
- combined features не улучшают качество относительно отдельных групп;
- значит в текущем протоколе добавочная сила FinBERT-признаков не подтверждается;
- это не опровергает потенциальную полезность LLM features вообще, но ограничивает вывод рамками AAPL, периода, таргета и split.
