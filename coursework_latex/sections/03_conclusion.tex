\section{Заключение}

В данной работе рассматривалось влияние LLM/FinBERT-generated features на прогнозирование финансовых временных рядов. В отличие от постановки, где основной целью является поиск модели с максимальной точностью, работа была построена вокруг анализа признаков. Основной исследовательский вопрос состоял в том, дают ли признаки, извлечённые из финансовых текстов с помощью языковой модели, самостоятельный сигнал и добавочную предсказательную силу относительно рыночных признаков.

В теоретической части были рассмотрены три группы работ. Первая связана с прогнозированием финансовых временных рядов и ролью информации в динамике цен \cite{fama1970efficient}. Вторая показывает, что финансовые тексты, новости и социальные медиа могут быть источником признаков, отражающих ожидания и настроения участников рынка \cite{tetlock2007media,loughran2011liability,kearney2014textual,bollen2011twitter}. Третья связана с развитием языковых моделей: Transformer и BERT сделали возможным получение контекстных текстовых представлений, а FinBERT адаптирует этот подход к финансовому языку \cite{vaswani2017attention,devlin2019bert,araci2019finbert,yang2020finbert}. На основе этого был обоснован feature-centric подход: языковая модель используется не как самостоятельный прогнозист, а как генератор признаков для downstream-модели.

Эмпирическая часть была построена как серия последовательных экспериментов. RUN 03 проверял базовую постановку на AAPL с тремя группами признаков:

\[
X_{price} = \{p_t\}, \qquad X_{text} = \{e_t\}, \qquad X_{combined} = \{[p_t;e_t]\},
\]

где $p_t$ --- рыночные признаки, $e_t$ --- FinBERT embedding, а $[p_t;e_t]$ --- объединённый вектор признаков. RUN 04 расширял рыночный baseline за счёт rolling- и volume-based признаков и проверял, сохраняется ли вклад FinBERT embeddings после PCA-сжатия. RUN 05 расширял проверку на crypto-news, несколько криптоактивов и горизонты 1, 3 и 7 дней.

Результаты RUN 03 показали наличие слабого самостоятельного текстового сигнала. Модель \texttt{FinBERT-only} получила accuracy 0.550 против 0.532 у majority baseline:

\[
\Delta_{FinBERT-majority}^{Accuracy} = 0.550 - 0.532 = 0.018.
\]

Добавление FinBERT embeddings к price baseline также улучшало слабую price-only модель по accuracy и balanced accuracy:

\[
\Delta_{text}^{Accuracy} = 0.486 - 0.410 = 0.076,
\]

\[
\Delta_{text}^{Balanced\ Accuracy} = 0.474 - 0.400 = 0.074.
\]

Однако этот результат не был устойчивым: объединённая модель не превосходила majority baseline по accuracy, уступала \texttt{FinBERT-only}, а ROC-AUC оставался ниже случайного уровня. Поэтому RUN 03 дал не сильноположительный, а скорее слабоположительный и методологически осторожный вывод.

RUN 04 усилил этот вывод. После добавления rolling- и volume-based признаков лучшей группой стала \texttt{market\_full}: accuracy составила 0.560, balanced accuracy --- 0.577, ROC-AUC --- 0.853. Добавление PCA-сжатых FinBERT embeddings к \texttt{market\_full} не улучшило accuracy и balanced accuracy, а ROC-AUC немного снизился с 0.853 до 0.846. Следовательно, в более строгом сравнении для AAPL основной вклад давали расширенные рыночные признаки, а устойчивая добавочная предсказательная сила FinBERT embeddings не подтверждалась.

RUN 05 показал, что этот отрицательный вывод нельзя автоматически обобщать на все активы и источники данных. В мультиактивной crypto-постановке текстовые признаки дали заметный прирост для отдельных активов. Наиболее выраженный эффект получен для BNB: на горизонте 1 день добавление FinBERT-derived и text-meta признаков к \texttt{market\_full} повысило balanced accuracy с 0.498 до 0.615, ROC-AUC --- с 0.476 до 0.670, accuracy --- с 0.494 до 0.612. На горизонте 7 дней для BNB balanced accuracy выросла с 0.415 до 0.569. Для SOL прирост balanced accuracy был положительным на всех трёх горизонтах, хотя менее устойчив по ROC-AUC. Для BTC эффект оказался слабым и почти исчезал на горизонте 7 дней.

Главный вывод работы можно сформулировать следующим образом. LLM/FinBERT-generated features не дают универсального преимущества относительно сильного рыночного baseline, но их вклад не является нулевым. В одних постановках, например для AAPL при расширенном market baseline, добавочная сила текстовых признаков не подтверждается. В других постановках, прежде всего для BNB на crypto-news, текстовые признаки дают существенный прирост качества. Следовательно, полезность LLM-generated features зависит от актива, источника текстов, горизонта прогноза, способа построения текстовых признаков и строгости baseline-сравнения.

Полученный результат согласуется с современной литературой по текстовому и мультимодальному финансовому прогнозированию: текстовые признаки могут быть полезны, но их эффект зависит от источника данных, временного выравнивания, таргета, горизонта прогноза и качества экспериментального протокола \cite{hiew2019bert,zou2022prebit,jiang2023finbert,albada2025bertweet,vuong2025llm}. Поэтому дальнейшая работа должна быть направлена не только на усложнение моделей, но и на более точную проверку признаков.

К возможным направлениям продолжения исследования относятся:

\begin{enumerate}
    \item расширение эксперимента на большее число акций, криптоактивов и рыночных режимов;
    \item проверка других горизонтов прогноза, включая внутридневные и многодневные горизонты;
    \item прямое сравнение FinBERT embeddings, sentiment-score, topic features и emotion features;
    \item использование rolling-window validation вместо одного фиксированного split;
    \item проверка статистической устойчивости эффекта признаков на нескольких временных периодах;
    \item анализ отдельных новостных событий, чтобы понять, какие типы текстов дают наибольший вклад в прогноз.
\end{enumerate}

Таким образом, работа показывает, что LLM/FinBERT-generated features являются теоретически обоснованным источником признаков для финансового прогнозирования, но их полезность должна проверяться через строгие baseline-сравнения. Итоговый результат является смешанным: текстовые признаки не дают универсального улучшения, однако в отдельных условиях, особенно для BNB в RUN 05, они демонстрируют заметную добавочную предсказательную силу.
