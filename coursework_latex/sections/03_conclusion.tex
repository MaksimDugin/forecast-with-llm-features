\section{Заключение}

В данной работе рассматривалось влияние LLM/FinBERT-generated features на прогнозирование финансовых временных рядов. В отличие от постановки, где основной целью является поиск модели с максимальной точностью, работа была построена вокруг анализа признаков. Основной исследовательский вопрос состоял в том, дают ли признаки, извлечённые из финансовых текстов с помощью языковой модели, самостоятельный сигнал и добавочную предсказательную силу относительно рыночных признаков.

В теоретической части были рассмотрены несколько направлений литературы. Классическая финансовая теория подчёркивает, что цены быстро отражают доступную рынку информацию \cite{fama1970efficient}. Работы по финансовой текстовой аналитике показывают, что новости, корпоративные тексты и социальные медиа могут содержать признаки, связанные с настроениями, ожиданиями и вниманием участников рынка \cite{tetlock2007media,loughran2011liability,kearney2014textual,bollen2011twitter}. Развитие Transformer-архитектур и BERT-подхода сделало возможным получение контекстных текстовых представлений, а FinBERT адаптирует эту идею к финансовому языку \cite{vaswani2017attention,devlin2019bert,araci2019finbert,yang2020finbert}. На основе этого был обоснован feature-centric подход: языковая модель используется не как самостоятельный прогнозист, а как генератор признаков для downstream-модели.

Эмпирическая часть была построена как серия последовательных экспериментов. Основной эксперимент RUN 03 проверял базовую постановку на AAPL с тремя группами признаков:

\[
X_{price} = \{p_t\}, \qquad X_{text} = \{e_t\}, \qquad X_{combined} = \{[p_t;e_t]\},
\]

где $p_t$ --- рыночные признаки, $e_t$ --- FinBERT embedding, а $[p_t;e_t]$ --- объединённый вектор признаков. Дополнительный эксперимент 1 RUN 04 расширял рыночный baseline за счёт rolling- и volume-based признаков и проверял, сохраняется ли вклад FinBERT embeddings после PCA-сжатия. Дополнительный эксперимент 2 RUN 05 расширял проверку на \texttt{crypto-news}, несколько криптоактивов и горизонты 1, 3 и 7 дней.

Результаты основного эксперимента RUN 03 показали наличие слабого, но неустойчивого текстового сигнала. Модель \texttt{FinBERT-only} получила accuracy 0.550 против 0.532 у \texttt{majority baseline}:

\[
\Delta_{FinBERT-majority}^{Accuracy} = 0.550 - 0.532 = 0.018.
\]

Добавление FinBERT embeddings к price baseline также улучшало слабую модель \texttt{prices-only} по accuracy и balanced accuracy:

\[
\Delta_{text}^{Accuracy} = 0.486 - 0.410 = 0.076,
\]

\[
\Delta_{text}^{Balanced\ Accuracy} = 0.474 - 0.400 = 0.074.
\]

Однако этот результат не был устойчивым: объединённая модель не превосходила \texttt{majority baseline} по accuracy, уступала \texttt{FinBERT-only}, а ROC-AUC оставался ниже 0.5. Поэтому основной эксперимент RUN 03 дал не сильноположительный, а методологически осторожный вывод: текстовый сигнал присутствует слабо, но его комплементарность с базовыми рыночными признаками не подтверждается.

Дополнительный эксперимент 1 RUN 04 усилил этот вывод. После добавления rolling- и volume-based признаков лучшей группой стала \texttt{market\_full}: accuracy составила 0.560, balanced accuracy --- 0.577, ROC-AUC --- 0.853. Добавление PCA-сжатых FinBERT embeddings к \texttt{market\_full} не улучшило accuracy и balanced accuracy, а ROC-AUC немного снизился с 0.853 до 0.846. Следовательно, в более строгом сравнении для AAPL основной вклад давали расширенные рыночные признаки, а устойчивая добавочная предсказательная сила FinBERT embeddings не подтверждалась.

Дополнительный эксперимент 2 RUN 05 показал, что отрицательный вывод по AAPL нельзя автоматически обобщать на все активы, источники данных и горизонты прогноза. В мультиактивной crypto-постановке текстовые признаки дали заметный прирост для отдельных активов. Наиболее выраженный результат получен для BNB: на горизонте 1 день добавление FinBERT-derived и text-meta признаков к \texttt{market\_full} повысило balanced accuracy с 0.498 до 0.615, ROC-AUC --- с 0.476 до 0.670, accuracy --- с 0.494 до 0.612. На горизонте 7 дней для BNB balanced accuracy выросла с 0.415 до 0.569. Для SOL прирост balanced accuracy был положительным на всех трёх горизонтах, хотя менее устойчив по ROC-AUC. Для BTC эффект оказался слабым и почти исчезал на горизонте 7 дней.

Главный вывод работы можно сформулировать следующим образом. LLM/FinBERT-generated features не дают универсального преимущества относительно сильного рыночного baseline, но в отдельных условиях демонстрируют добавочный прогнозный сигнал. В одних постановках, например для AAPL при расширенном \texttt{market\_full} baseline, добавочная сила текстовых признаков не подтверждается. В других постановках, прежде всего для BNB на \texttt{crypto-news}, текстовые признаки дают существенный прирост качества. Следовательно, полезность LLM-generated features зависит от актива, источника текстов, горизонта прогноза, способа построения текстовых признаков и строгости baseline-сравнения.

При этом результат BNB не следует интерпретировать как окончательное доказательство универсальной эффективности текстовых признаков. RUN 05 имеет exploratory характер: в нём одновременно сравниваются несколько активов, горизонтов и групп признаков. Для BNB 1d paired bootstrap по датам дал положительные 95\% confidence intervals для прироста balanced accuracy и ROC-AUC, а prediction-level permutation sanity check показал, что фактический прирост выше почти всех случайно перемешанных вариантов. Это усиливает интерпретацию BNB 1d как кандидата на наличие текстового сигнала, но не снимает необходимость дальнейшей проверки на независимом периоде, в rolling-window схеме и с учётом множественного сравнения.

Полученный результат согласуется с современной литературой по текстовому и мультимодальному финансовому прогнозированию: текстовые признаки могут быть полезны, но их эффект зависит от источника данных, временного выравнивания, целевой переменной, горизонта прогноза и качества экспериментального протокола \cite{hiew2019bert,zou2022prebit,jiang2023finbert,albada2025bertweet,vuong2025llm}. Поэтому дальнейшая работа должна быть направлена не только на усложнение моделей, но и на более точную проверку признаков.

К возможным направлениям продолжения исследования относятся:

\begin{enumerate}
    \item расширение эксперимента на большее число акций, криптоактивов и рыночных режимов;
    \item проверка других горизонтов прогноза, включая внутридневные и многодневные горизонты;
    \item прямое сравнение FinBERT embeddings, sentiment-score, topic features, emotion features и text-meta признаков;
    \item использование rolling-window validation вместо одного фиксированного разбиения;
    \item проверка статистической устойчивости эффекта признаков через bootstrap confidence intervals для всех активов и горизонтов;
    \item sanity check через перестановку дат текстовых признаков с переобучением модели;
    \item анализ отдельных новостных событий, чтобы понять, какие типы текстов дают наибольший вклад в прогноз.
\end{enumerate}

Таким образом, работа показывает, что LLM/FinBERT-generated features являются теоретически обоснованным источником признаков для финансового прогнозирования, но их полезность должна проверяться через строгие baseline-сравнения. Итоговый результат является смешанным: текстовые признаки не дают универсального улучшения, однако в отдельных условиях, особенно для BNB в дополнительном эксперименте RUN 05, они демонстрируют заметную добавочную предсказательную силу.
